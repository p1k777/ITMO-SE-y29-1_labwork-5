## Структура проекта

- `instruments` - реализация инструментов (штуки пишут сырой звук в дорожки, которые хранятся в них же)
- `effects` - эффекты, применяются к уже готовому звуку из дорожек инструментов
- `fileparser.*` - парсит ITMO Loops файл, создает объекты необходимые для генерации звука
- `wavwriter.*` - пишет дорожки из инструментов в `.wav`
- `defines.*` - объявления и определения полезных функций
- `notes.txt` - важный файл с частотами нот
